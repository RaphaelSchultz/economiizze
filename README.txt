Economiizze — Landing Page
--------------------------------
Arquivos:
- index.html
- styles.css
- assets/logo.png

Como rodar localmente:
1) Baixe o .zip e extraia.
2) Dê duplo clique em index.html (abre no navegador).

Como publicar no GitHub Pages:
1) Crie um repositório (ex.: economiizze-landing).
2) Envie os arquivos (index.html, styles.css e pasta assets).
3) Em Settings > Pages, escolha a branch `main` e a pasta `/root` (ou `docs` se preferir).
4) Salve. Em alguns minutos o site estará no ar.

Links de CTA já apontam para: https://whats.ly/grupoeconomiizze

Quer mudar a imagem?
- Substitua assets/logo.png pelo seu arquivo (mesmo nome) ou ajuste o caminho em index.html.
